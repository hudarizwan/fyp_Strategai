# Environment Setup Guide

## Required variables

- `SUPABASE_URL`
- `SUPABASE_KEY`
- `SUPABASE_SERVICE_KEY`
- `SUPABASE_DB_URL`
- `OLLAMA_BASE_URL`
- `OLLAMA_MODEL`
- `VITE_API_BASE_URL`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Local setup

```bash
copy .env.example .env
copy backend\.env.example backend\.env
copy frontend\salik-frontend\.env.example frontend\salik-frontend\.env
```

Fill in the real values before running the stack.
