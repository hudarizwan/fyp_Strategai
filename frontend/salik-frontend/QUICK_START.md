# 🚀 Quick Start Guide

## Prerequisites
- Node.js 18+ and npm (comes with Node.js)

## Installation & Run (3 Steps)

### 1. Install Node.js
Download from: **https://nodejs.org/** (LTS version)

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

**That's it!** Open http://localhost:5173 in your browser.

---

## Using the Setup Script (Windows)

If you're on Windows, you can use the automated setup script:

```powershell
.\setup.ps1
```

This will check for Node.js and install all dependencies automatically.
